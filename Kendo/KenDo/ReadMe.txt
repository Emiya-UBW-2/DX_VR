剣道 -つるぎ みち- Ver 1.0.0　　　　　　笑也 知郎

このアプリケーションについて
　　剣道を再現した1人向けアクションです。
　　微妙に連盟の定めるものとは異なりますが、中高剣道を基にしたシステムとなっています。
　　チュートリアルにはおおむね5分、試合は2本先取の3分勝負で延長戦はありません。

起動方法
	ダウンロードしたZipを解凍し、exeを実行してください。
	シェーダーバイナリによりウイルス警告が入る場合があります(お使いの環境に影響を及ぼすファイルはございません)

！「DirectXのバージョンが適用していません」が出る場合！
	お使いの環境がDirectX11に対応していません、設定からDirectX9.0cを選択し、再起動してください。

！挙動が重い場合！
	設定からグラフィックプリセットまたはレンダリングスケールを下げていただくことで解消されます。

！ゲームパッドが正常に機能しない場合！
	設定からPadTypeをDirect Input/X Inputどちらかに切り替えてお試しください。

推奨動作環境(開発環境に同じ、最高のグラフィック設定でも60FPSを維持して動作します)
	Windows10
	AMD Ryzen 7 2700X Eight-Core Processor
	RAM 16GB
	NVIDIA GeForce GTX 1660 Ti

推奨動作環境(開発環境に同じ、最低のgraphic設定で動作を最低限確認できている環境です)
	Windows7
	Core-i7 2670QM
	RAM 8GB
	オンボードグラフィック

操作
　　PS4/Xbox互換ゲームパッドに対応しています。パッドが刺さっている場合そちらでの操作が優先されます。(抜き差し対応、指していないと自動でキーボード操作になります)
　　詳しい操作は各画面左下のアイコンを参照ください。
　　どんな場面でもESCキーで停止確認を行います。
