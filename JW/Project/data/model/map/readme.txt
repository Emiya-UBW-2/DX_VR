<Tips>
　最初は建物内の入れ替えを予定していたのですが、ボーンがえらいことになるので分割しました。
　キルハウスは通路を進んでいくタイプで、FPSのチュートリアルとかにありそうなあれっぽいそれです。
　オサレキルハウスは9つの部屋をクリアしていくそれっぽいあれです。マンターゲットはすべて倒れます。
　インドアフィールドはサバゲのインドアフィールドっぽいそんな感じのあれです。
　シューティングレンジが一番マシだと思います。　

　各モデルとも、ガラスはアクセサリ読み込みで対応しています。
　「キルハウス」「オサレキルハウス」「インドアフィールド」は、客席ガラス手前.xと客席ガラス奥.xを読み込んだ後、センターに関連付けてください。
　「シューティングレンジ」は、レンジガラスA.x,B.xをセンターに、レンジドアガラスL.x,R.xをそれぞれ左右のドアに関連付けてください。
　半透明なので順番にご注意ください。


<おまけについて>
　テクスチャは下層のtexフォルダから読み込むので、texフォルダをコピーするなり.xファイルを移動するなりしてください。
　的は片面1:2、レンジ的は3:4となっています。
　左半分が表面、右半分が裏面となります。
　多分アルファを付与した個所は透明になります。
　あと厳密に綺麗なUV展開ではないのでご了承ください。


<改造について>
　好き勝手に改造して使ってください。
　レイアウトを変えるだけならインドアフィールドが室内を空っぽにできるので使いやすいと思います。
　改造品であれば配布は自由に行っていただいて構いません。
　その際には出典元を明記していただけるとありがたいです。
　連絡は基本的に不要ですが、頂けたら嬉しいです。
