■ 特記

　このモデルは基本的にオリジナルですが、胴体に関しては大和工廠P様に大幅な修正（というか事実上、作り直し）をしていただいたものを基本として制作しております。従いまして、日本国著作権法に定める一次著作権が、私だけでなく大和工廠P様にも発生しています。
　本モデルに関するご連絡・ご相談（例：改造して配布したい、などの相談）は私が一括してとりまとめますが、改造を考えておられる方は、ご相談の前に上記の事項を頭に入れておいてください。

　非常識な利用をしないで下さい。他の人や私が不快に思うような利用をしないで下さい。

■ ファイルについて

　ボーンを全てkeynoteで定義し、メタセコ上で動作確認を行ってからpmxエクスポートし、PMXEditorで整形してあります。よって、同梱のmqoファイルを完全な意味で確認するためには、keynoteプラグインが必須となります。

　.csvファイルはPMXEditor用です。AH-6E用のボーン・表情枠はMH-6Eと同じですので、そちらを流用して下さい。

■ さいごに

　モデルを使用した動画・静画を製作・発表された場合、取説動画を親作品に指定していただけますと助かります。

　資料代で月末はリアルに食費すら事欠くことがあります。かわいそうに思った場合、ウェブサイト記載の方法で手を差し伸べていただけますと、次のモデルに反映させることができます。
