H&K MP5　セットVer.1.11

いたってシンプルなMP5K・・・と折り畳みストックに差し替えたMP5A2とM203グレネードランチャーを取り付けた無茶振り
MP5のセットです。
※バースト付いたA４版モデルも追加しました。
※さらにスライドストック型のA3・A5版も追加　バースト有がA5です　もちろんM203付モデルもいれました。

おまけもあるよ！

今回もモーフで稼動部を動かす前提で作られています。（マガジン・リアサイト・スリング部を除く）
もちろんボーン操作でも動かせますが連動しない箇所があるのであまりおすすめはしません。

おまけパーツ（MP5K用フラッシュハイダー・MP5K用ストック・ローマウント・ハイマウント）は
外部親で本体のセンターに追従させるだけで大丈夫です。
マガジン系パーツは本体のマガジンボーンに追従させて、本体のマガジンはマガジン消モーフで消してしまえばOK！

面倒であればPMXEで合成してください。
